Ik heb slechts een beperkt aantal binnen op de MCU gebruikt, maar de ATtiny1616 is een betaalbare en mooi kleine MCU met een prettige hoeveelheid geheugen.
De volgende pinnen zijn in gebruik:
11   (PC2)  Voor het snoer met 43 neoPixel leds
A2   (PC2)  Voor het knopje, staat standaard op hoog, en wanneer je deze indrukt gaat die op laag.
10   (PC0)  Gaat via een mosfet naar het middelste padje op de bovenste rij (achter) met als doel mogelijk een passieve buzzer op aan te sluiten
 8   (PB2)  Gaat direct naar het kleine linker padje op de bovenste rij (achter) ook met als doel een passieve buzzer op aan te sluiten. Beide om te kunnen testen

 UPDI is via een weerstand verbonden met het middelste pad op de onderste rij (achterkant) zodat deze direct op pin 6 van een arduino kan worden aangesloten voor JTAG2UPDI programmeren (de extra weerstand is dus niet nodig
 VCC  Kan worden gevoed het grote pootje (rechts op de voorkant en links op de achterkant) en ook met het padje links op de onderste rij (achter) om makkelijker te kunnen verbinden tijdens het programmeren
 GND  Kan worden gevoed via het kleine pootje (links op de voorkant en rechts op de achterkant) en ook via het padje rechts op de onderste rijd (achter) ook weer om makkelijker te kunnen verbinden met programmeren.
